# GoLink

A decentralized P2P file and message sharing application built with Go.

## Concept

[This project is developed as a university assignment for DevOps course.]

## Features (Planned)

*   P2P file transfer
*   Encrypted messaging
*   LAN discovery
*   NAT Traversal

## How to build

1.  Clone the repo
2.  Run `go build -o golink cmd/main.go`
